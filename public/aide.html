📖 MODE D'EMPLOI

🎡 Fancy Fête Hell Beck

Gestion des commandes en temps réel entre l'accueil et la cuisine.

📝 1. À l'Accueil

  • Prendre commande : Cliquez sur un plat. Choisissez Taille, Cuisson et Accompagnement.
  • Modifier les quantités : Dans le plateau en cours, utilisez les boutons et pour ajuster chaque ligne.
  • Envoi Cuisine : Sélectionnez une table (1-16) puis validez. Les frites seules passent en ⚡ EXPRESS.
  • Message Flash : Utilisez la barre grise en haut pour envoyer un texte urgent qui s'affichera en grand en cuisine.

👨‍🍳 2. En Cuisine

  • Barre de production : Regardez en haut pour voir le cumul total de viandes et frites à cuire.
  • Code couleur d'attente :
    • Orange (10 min) : Attention, ça commence à être long.
    • Rouge clignotant (15 min) : URGENCE, le client attend trop !
  • Terminer : Cliquez sur "TERMINÉ ✅" pour prévenir l'accueil.

🏁 3. Livraison & Retour

  • Bulles Vertes : Elles s'empilent en haut à droite dès qu'un plateau est prêt. Cliquez pour vérifier.
  • Libération : La table est libérée quand vous cliquez sur "VÉRIFIÉ / SERVI ✅".
  • Forcer une table : Si une table reste bloquée (X), faites un appui long (2 sec) sur le numéro pour la libérer manuellement.
