📖 Manuel d'Utilisation - Caisse Resto

ACCUEIL Prise de commande

1. Sélection : Cliquez sur un plat. Choisissez la taille (👨/🧸) et la cuisson si demandée.
2. Options : L'accompagnement (🥗/🍎) est automatique pour certains plats, sinon choisissez-le.
3. Remarques : Tapez les allergies ou demandes spéciales (ex: "SANS OIGNONS") en bas.
4. Validation : Cliquez sur le numéro de table (1-16) puis sur 🚀 ENVOYER.

💡 Cas particulier : Les FRITES sont "Express". Pas besoin de numéro de table, elles s'envoient directement !

CUISINE Préparation

1. Tickets : Les commandes apparaissent avec un chrono. Les menus 🧸 ENFANT sont en haut du ticket.
2. Barre de prod : En haut, vous voyez le total cumulé de tous les plats restants à cuire.
3. Alerte : Si un ticket clignote en ROUGE, il est en retard (+15 min) !
4. Validation : Une fois le plateau prêt, cliquez sur PRÊT ✅.

SERVICE Livraison

1. Notification : Une bulle verte (ou bleue) apparaît en haut à droite de l'accueil.
2. Vérification : Cliquez sur la bulle pour vérifier le contenu du plateau.
3. Libération : Cliquez sur VÉRIFIÉ / SERVI pour libérer la table et archiver la vente.
